## 颜色名

```css
color: red;
background-color: blue;
```

## RGB / RGBA

```css
color: rgb(255, 255, 255);          /* 白色 */
color: rgba(255, 0, 0, 0.5);        /* 半透明红色 */
```

## Hex（十六进制）

```css
color: #fff;      /* 白色（缩写） */
color: #ffffff;   /* 白色（完整） */
color: #000;      /* 黑色 */
```

## HSL / HSLA

```css
color: hsl(0, 100%, 50%);            /* 红色 */
color: hsla(0, 100%, 50%, 0.5);      /* 半透明红色 */
```

## 元素透明度

```css
opacity: 0.5;    /* 整个元素半透明（含子元素），值 0–1 */
```

> 注意：`opacity` 影响整个元素（含子元素），如果只想让背景透明，用 `rgba()` 或 `hsla()` 设置 `background-color`。
