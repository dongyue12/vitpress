## Md 语法其实很简单
主要就是：
1. [标题](#标题)
2. [列表](#列表)
3. [链接](#链接)
4. [代码块](#代码块)
5. [引用](#引用)
6. [表格](#表格)
7. [文字样式](#文字样式)
### 标题
标题分为6个级别，使用#号表示，如下：
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 列表
列表分为有序列表和无序列表，如下：
```
# 无序列表
- 第一项
- 第二项
- 第三项

# 有序列表
1. 第一项
2. 第二项
3. 第三项

# 嵌套列表
- 第一项
  1. 第二项
    - 第三项

# 代办列表
- [ ] 第一项
- [x] 第二项
- [ ] 第一项
```

### 链接
链接分为普通链接和图片链接，如下：
```
# 普通链接
[百度](https://www.baidu.com)

# 锚点链接
[百度](#百度)

# 图片链接
![图片描述](https://www.baidu.com/img/bd_logo1.png)
```

### 代码块
代码块分为行内代码块和代码块，如下：
````
# 行内代码块
`代码`

# 代码块
```html
代码
```
````

### 引用
引用分为普通引用和引用链接，如下：
```
# 普通引用
> 引用内容

# 层级
> 引用内容
>> 引用内容
>>> 引用内容

# 引用链接
> 引用内容 [链接](https://www.baidu.com)
```

### 表格
表格分为普通表格
```
# 普通表格
| 表头1 | 表头2 | 表头3 |
| ----- | ----- | ----- |
| 内容1 | 内容2 | 内容3 |
| 内容1 | 内容2 | 内容3 |

# 修改表格对齐方式，使用:表示对齐方式，如下：
# 居中对齐：
| :---: | :---: | :---: |

# 左对齐：
| :--- | :--- | :--- |

# 右对齐：
| ---: | ---: | ---: |
```

### 文字样式
```
# 加粗
**加粗**

# 斜体
*斜体*

# 下划线
<u>下划线</u>

# 删除线
~~删除线~~

# 高亮
==高亮==

# 上标
X^2^

# 下标
H~2~O
```
