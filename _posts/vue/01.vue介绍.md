## vue3和vue2的区别
1. vue3 主要更新的是组合式 API和脚手架切换为 [vite](https://vitejs.cn/vite3-cn/guide/)，实现了速度的提升
2. 和vue2的选项式 API ，vue3的代码量更少，性能更高，体积更小
3. 现阶段已经全部转移到vue3了，所以本文着重介绍vue3
4. 具体查看[vue官网](https://cn.vuejs.org/)的信息

## 第一步显示安装Vue3

1. 更具包管理器选择你喜欢的，这里我推荐使用bun
2. bun是用rust+zed写的，速度非常快，官方说比node块3倍

```bash
# bun 的安装方式为
npm install -g bun  

# 接下来安装vue
bun create vue@latest
```

## 安装时问几个问题
1. 选择是否使用typeScript，这里更具自己的情况选择
2. 其他选项选不选都可以
