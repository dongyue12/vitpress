## 条件语句

### if / else if / else

```javascript
let x = 10;
if (x > 10) {
    console.log("x > 10");
}
else if (x < 10) {
    console.log("x < 10");
}
else {
    console.log("x = 10");
}
```

### 三元运算符

```javascript
let y = 10;
console.log(y > 10 ? "y > 10" : "y <= 10");
```

### switch 语句

```javascript
let z = 10;
switch (z) {
    case 10:
        console.log("z = 10");
        break;
    case 20:
        console.log("z = 20");
        break;
    default:
        console.log("z = other");
        break;
}
```

## 循环语句

### for 循环

```javascript
for (let i = 0; i < 10; i++) {
    console.log(i);
}
```

### while 循环

```javascript
let j = 0;
while (j < 10) {
    console.log(j);
    j++;
}
```
